# 자바 스크립트 - 자료형

### 변수

```javascript
var a = '';
var b = 0;
var c = false;
var d = null;
var e = undefined;
var f = [];
var g = {};
var h = function() {};
```

### 자료형

#### 문자형(String)
```javascript
var string = "string";
var string2 = 'string';
var string3 = "'string'";
var string4 = '"string"';
var string5 = '\'string\'';
```

#### 숫자(Number)
```javascript
var number = 1;
var float = 5.6;
```

#### 불린(Boolean)
```javascript
var bool = true;
var boot2 = false;
```

#### Undefined, Null
```javascript
var a;
a; //undefined

var b = 125;
b = null;
b;

a == b // true
a === b // false
```
